## Apple Sales Global Analysis | Power BI

# Project Overview
This project provides a comprehensive look at global sales trends, product performance, and operational metrics for an Apple Sales database. The goal was to transform raw sales data into actionable business insights using Power BI.

# Key Features
Executive Summary: High-level KPIs showing $6.17bn Total Sales, $1.1bn YTD, and 5.72M Units Sold.

Geographical Insights: A Treemap visualization showing the USA ($1.23bn) as the leading market, followed by Australia and China.

Product Analytics: Breakdown of sales by category (Tablets and Accessories leading) and a focused look at the "Oldest 10 Products."

Operational Tracking: A donut chart analysis of Repair Status, showing a balanced distribution between Completed, Pending, In-Progress, and Rejected cases.

Time Intelligence: Analysis of sales trends by month and year (2020–2024).

# Tech Stack & Skills
Power BI: Data visualization and dashboard layout design.

DAX (Data Analysis Expressions): Created measures for YOY (Year-over-Year), MTD (Month-to-Date), and LY (Last Year) comparisons.

Data Modeling: Established relationships between sales, products, stores, and geography tables.

Power Query: Cleaned and transformed data for accurate reporting.
